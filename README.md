# Sistema de Processo Seletivo – C#

Projeto desenvolvido para a disciplina de Algoritmos e Estruturas de Dados.

O sistema realiza a classificação de candidatos com base na média das notas e os distribui entre cursos conforme o número de vagas, gerando também filas de espera.

## Funcionalidades
- Leitura de arquivo de entrada
- Cálculo de média dos candidatos
- Ordenação utilizando Quicksort
- Critério de desempate por redação
- Seleção por primeira e segunda opção
- Geração de arquivo de saída

## Estruturas utilizadas
- Dictionary
- List
- Fila Flexível (implementada manualmente)

## Tecnologias
- C#
- .NET Console
